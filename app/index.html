Mega menus

Flust.no vs Adobe accessible mega menu

Simplified example

<ol>
  <li>
    <h4><a href="" id="menu-item-1-header" aria-haspopup="true" aria-owns="menu-item-1-content" aria-expanded="false">Menu item 1</a></h4>
    <div id="menu-item-1-content" role="group" aria-expanded="false" aria-hidden="true" aria-labelledby="menu-item-1-header">
      <p>Megamenu content</p>
    </div>
  </li>
</ol>

Note: aria-expanded and aria-hidden have to be changed using javascript when toggling the menu. Also, keyboard navigation should be handled.
